# Insubuy Code Challenge
### Steps to run this code

1. Download or clone this repository
   `git clone git@github.com:ronaktk/Insubuy-Code-Challenge.git`
2. `cd Insubuy-Code-Challenge`
3. `npm install`
4. `npm start`
5. `cd client`
6. `npm install`
7. `npm start`
